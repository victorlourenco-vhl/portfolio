Sobre

Atualmente busco minha primeira oportunidade na área de desenvolvimento de software. Estou cursando Análise e Desenvolvimento de Sistemas pela Fatec Ipiranga (2022 - 2025), curso este, o qual escolhi por achar incrível o modo como a tecnologia impacta na vida das pessoas.

Eu gosto muito de atuar com Desenvolvimento Full Stack utilizando Git, Github, Python, HTML, CSS e JavaScript. Atualmente estou estudando Banco de dados Relacionais com MySQL, Django(para Projetos Web mais complexos), Desenvolvimento Mobile com React Native e desenvovimento Desktop com PyQt5 juntamente com QtDesigner. Acho muito importante ser um profissional com diferentes habilidades para compreender e contribuir com a equipe da melhor forma possível.
